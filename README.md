# Car Sales Dashboard

An interactive and data-driven Power BI dashboard designed to track, analyze, and visualize key performance indicators (KPIs) for car sales. This project utilizes SQL for backend data extraction/transformation and Power BI for frontend data visualization and insights generation.

---

## 🚀 Project Overview

The objective of this project is to provide stakeholders with a comprehensive view of car sales performance, allowing them to track revenue, identify sales trends across different car types and body styles, and optimize inventory management.

### Key Insights Tracked:
* **Sales Performance Over Time:** Year-to-Date (YTD) and Month-to-Date (MTD) sales comparison.
* **Regional Analytics:** Identifying high-performing regions and dealer locations.
* **Product Segmentation:** Sales breakdown by car manufacturer, model, body style, and color.
* **Interactive Filters:** Slicers for dynamic filtering by date, region, and vehicle specifications.

---

## 🛠️ Tech Stack & Tools

* **Database / ETL:** SQL (Data extraction, cleaning, and KPI aggregation)
* **Visualization:** Power BI Desktop (Data modeling, DAX measures, and Report design)
* **Dataset:** Car sales historical transactional data

## 📐 Key Metrics & DAX Formulas

Here are some of the core metrics implemented in the dashboard:

* **YTD Total Sales:** Cumulative sales from the beginning of the current year.
* **MTD Total Sales:** Cumulative sales for the current month.
* **Year-over-Year (YoY) Growth:** Tracking performance shifts compared to the previous year.
